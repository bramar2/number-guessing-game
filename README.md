# number-guessing-game:  A number guessing game (CLI)
https://roadmap.sh/projects/number-guessing-game
## Code
CMake-based project.
## Building
On root, run the following commands:
- `cmake -S . -B build/`
- `cmake --build build/`


The executable should be in build/number-guessing-game.exe.
